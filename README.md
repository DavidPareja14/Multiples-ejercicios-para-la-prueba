# Multiples-ejercicios-para-la-prueba
Pongo a prueba mis habilidades con retos interesantes.

En el Ejercicio 1 se logra lo sigiente:
* Se puede entregar la máxima dimensión de un arreglo o matriz
* Si hay la misma cantidad de elementos en cada dimensión, se retorna True, de lo contrario, retorna False.
* Al final, no importa que tantas dimensiones existan, se puede retornar el valor de la suma de todos los números que integren el arreglo o matriz.

En el Ejercicio 2, se obtienen muy buenos resultados como:
* Verificar si una expresión aritmética en forma de cadena está bien escrita.
* Si la expresión está bien escrita, se realiza la operación matemática.
* Se permiten expresiones tipo 1 +2 -2 * 8 / ( 1 - 2) + 2, pero no se aceptan paréntesis anidados
